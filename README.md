# SAMSUNG HU6900-Series (UIRT Remote)
Basic remote for the SAMSUNG UE55HU6905 TV over USB-UIRT

## Features
* Volume (up, down, mute)
* Switch Input Source
* Power (off/on)

## Requirements
* A USB-UIRT device (http://www.usbuirt.com/)
